# 교회문서키트 BASIC 1.15

교회 실무자를 위한 문서 작성 웹앱입니다. 구매자 이메일 인증, PWA 설치 지원, PDF/PNG 저장 기능을 포함합니다.

## BASIC 1.15 판매 전 최종 정리판

이번 버전은 기능을 더 늘리기보다, 구매자가 실제로 사용할 때 헷갈릴 수 있는 부분을 줄이는 데 집중했습니다.

### 1. 브랜드/버전 표기 정리
- 앱 화면에는 버전 숫자를 과하게 노출하지 않고 `교회문서키트 BASIC` 중심으로 보이도록 정리했습니다.
- 내부 관리는 BASIC 1.15로 유지합니다.

### 2. 일정표 편의성 개선
- 편집판에서는 일정 카드가 자동으로 움직이지 않습니다.
- 필요할 때 `현재 일차 시간순 정리` 버튼으로 해당 일차 일정만 정리합니다.
- 출력 방식에 `한 페이지 맞춤`과 `일차별 여유형`을 추가했습니다.
- 4일 이상 일정은 `일차별 여유형`을 선택하면 A4 여러 장으로 나뉘어 더 읽기 좋습니다.

### 3. 부서 통합 주간보고서 입력 정리
- 부서/팀명은 입력하면 바로 반영됩니다.
- 엔터를 누르지 않아도 됩니다.
- 부서명을 지웠을 때 자동으로 임의 이름이 들어가지 않도록 정리했습니다.

### 4. 예산 입력 안정화
- 수입·지출 예산안은 4열 출력 형식을 유지합니다.
- 입력판은 접기/펼치기 카드형으로 유지해 항목이 많아져도 필요한 항목만 펼쳐 수정할 수 있습니다.
- 금액 칸에 `0`을 입력해도 직접 입력값으로 인정하도록 계산 방식을 보강했습니다.

### 5. 화면 가림 방지
- 작성기 주소 복사/로그아웃 안내 바가 저장 버튼이나 입력 영역을 가리지 않도록 하단 여백을 보강했습니다.
- 모바일에서도 하단 안내 바 때문에 마지막 입력칸이 가려지는 일을 줄였습니다.

### 6. 글자 크기 적용 보강
- 문서 기본도구의 글자 크기 설정이 본문뿐 아니라 표 내부 글씨에도 적용되도록 보강했습니다.
- 일정표는 A4 안에 들어가야 하므로, 일정이 많을 때는 자동 맞춤이 우선될 수 있습니다.

## 기본 제공 문서

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 부서별 주간보고서
4. 부서 통합 주간보고서
5. 행사 및 수련회 기획안

## 배포 방법

1. GitHub 저장소에 전체 파일을 교체 업로드합니다.
2. Vercel에서 자동 배포가 완료되는지 확인합니다.
3. 기존 Supabase/Vercel 환경변수는 그대로 유지합니다.

## 환경변수

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## 사용 안내

개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다. 공용 PC에서 사용한 경우에만 `공용 PC에서 로그아웃`을 눌러 주세요.

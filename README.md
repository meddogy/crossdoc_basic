# 교회문서키트 BASIC 1.4 완성 세트

교회 현장에서 반복적으로 사용하는 기본 문서를 빠르게 작성하고 PDF/PNG로 저장하는 웹앱입니다.

## 포함 문서

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 부서별 주간보고서
4. 부서 통합 주간보고서
5. 행사 및 수련회 기획안

## BASIC 1.4 주요 변경

- BASIC 문서 세트를 5종으로 확장했습니다.
- 기존 “교육부서 주간보고서”를 “부서별 주간보고서”로 범용화했습니다.
- 기존 “교육부 주간보고서”를 “부서 통합 주간보고서”로 범용화했습니다.
- 통합 주간보고서의 부서명을 직접 수정하고 부서/팀을 추가·삭제할 수 있습니다.
- 행사 및 수련회 기획안의 일정 입력을 1일차/2일차/3일차 탭 방식으로 정리했습니다.
- 기존 구매자 이메일 인증, PWA 설치 지원, 작성기 주소 복사 기능은 유지됩니다.

## 배포 방법

1. 이 폴더 전체를 GitHub 저장소에 업로드합니다.
2. Vercel에서 자동 배포를 확인합니다.
3. 기존 환경변수는 그대로 사용합니다.
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`

## 로그인 안내

구매자는 등록된 이메일로 로그인 링크를 받아 인증합니다.
개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다.
공용 PC에서 사용한 경우에만 “공용 PC에서 로그아웃”을 눌러주세요.

# 교회문서키트 BASIC 1.24

## 핵심 변경

- 모바일 안정 로그인 방식으로 변경했습니다.
- 기존 Magic Link 중심 안내를 줄이고, 이메일로 받은 6자리 인증코드를 앱 화면에 입력하는 흐름을 추가했습니다.
- PC와 모바일에서 각각 최초 1회 인증 후, 개인 기기에서는 로그아웃하지 않으면 세션이 자동 갱신됩니다.
- 기존 홈 대시보드, 내 문서 클라우드 저장, 베타 신청/관리 기능은 유지됩니다.

## 꼭 필요한 Supabase 설정

6자리 코드 방식이 제대로 작동하려면 Supabase 이메일 템플릿이 `{{ .Token }}`을 보여줘야 합니다.

Supabase 기본 메일 템플릿이 `{{ .ConfirmationURL }}`만 사용하는 상태라면 여전히 Magic Link가 발송될 수 있습니다. 이 경우 Custom SMTP를 설정한 뒤 이메일 템플릿을 코드 방식으로 바꿔 주세요.

권장 문구 예시는 `supabase_email_otp_template.txt` 파일을 참고하세요.

## 기존 베타테스터 안내

기존에 발송된 Sign in 링크는 무시해도 됩니다. 승인된 이메일 권한은 그대로 유지됩니다.

새 버전 배포 후에는 작성기 주소로 다시 접속하여 이메일을 입력하고, 새로 도착한 6자리 인증코드를 앱 화면에 입력하면 됩니다.

## 클라우드 저장 SQL

PC·모바일 이어쓰기 기능을 사용하려면 `supabase_cloud_documents.sql`을 Supabase SQL Editor에서 한 번 실행해야 합니다.
이미 실행했다면 다시 실행할 필요는 없습니다.

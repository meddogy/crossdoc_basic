# 교회문서키트 BASIC 1.10

교회 실무자를 위한 문서 작성 웹앱입니다. 구매자 이메일 인증, PWA 설치 지원, PDF/PNG 저장 기능을 포함합니다.

## BASIC 1.10 주요 개선

- 행사 및 수련회 기획안 일정표 입력 개선
  - 프리셋 버튼은 `당일 행사`, `1박 2일`, `2박 3일`, `3박 4일`까지만 노출합니다.
  - 4일차 이상은 `+ 일차 추가` 버튼으로 필요한 만큼 늘립니다.
  - 현재 일차에 새 일정을 추가하면 입력칸이 바로 보입니다.
  - 시작/종료 시간을 입력하면 문서 편집판에서도 시간순으로 정리됩니다.
  - 시간 입력칸은 클릭 시 전체 선택되어 기존 숫자가 남아 헷갈리는 문제를 줄였습니다.
  - `13:30`, `오후 1:30`, `1시 30분`처럼 입력하면 `13:30` 형식으로 정리됩니다.
- 부서 통합 주간보고서 개선은 유지됩니다.
  - 부서별 현황에서 부서명 추가, 삭제, 수정이 가능합니다.

## 기본 제공 문서

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 부서별 주간보고서
4. 부서 통합 주간보고서
5. 행사 및 수련회 기획안

## 배포 방법

1. GitHub 저장소에 전체 파일을 교체 업로드합니다.
2. Vercel에서 자동 배포가 완료되는지 확인합니다.
3. 기존 Supabase/Vercel 환경변수는 그대로 유지합니다.

## 환경변수

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## 사용 안내

개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다. 공용 PC에서 사용한 경우에만 `공용 PC에서 로그아웃`을 눌러 주세요.


## BASIC 1.10 변경사항
- 행사 및 수련회 기획안의 상세 일정표 시간을 직접 입력 방식에서 선택 방식으로 정리했습니다.
- 시작 시간을 선택하면 종료 시간이 기본 1시간 뒤로 자동 입력됩니다.
- 종료 시간이 시작 시간보다 빠르면 자동으로 보정됩니다.
- 일정을 추가한 뒤 시간을 고르면 문서 편집판에서도 시간순으로 정렬됩니다.


## BASIC 1.10 수정 내용
- 행사 및 수련회 기획안 일정표의 시작/종료 시간 선택 안정화
- 부서 통합 주간보고서 부서/팀명 입력을 한글 조합에 안정적으로 수정
- 부서별 주간 현황 표에서 긴 문장이 표 밖으로 넘어가지 않도록 줄바꿈 보강


## BASIC 1.10 수정 내용

- 수입계획/지출계획 입력을 좌우 스크롤 표에서 카드형 입력으로 변경했습니다.
- 수입 합계, 지출 합계, 차액을 상단 요약 카드로 바로 볼 수 있게 했습니다.
- 각 예산 항목은 항목명/내용/수량/단가/비고를 세로형 카드로 수정해 좁은 편집판에서도 잘리지 않게 했습니다.
- 기존 이메일 인증, PWA 설치, 작성기 주소 복사 기능은 그대로 유지합니다.

# 교회문서키트 BASIC 1.0 작성기

교역자와 교회 실무자가 자주 쓰는 3개 문서를 10분 안에 작성하고 PDF/PNG로 저장하는 상품화용 안정판입니다.

## 포함 문서

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 교육부서 주간보고서

## 사용 흐름

문서 선택 → 내용 작성 → 미리보기 확인 → PDF/PNG 저장

## 구매자 안내

- 외부 상품명은 `교회문서키트 BASIC 1.0 작성기`입니다.
- 내부 관리 버전은 `1.0.5`입니다.
- 복잡한 디자인 편집 프로그램이 아니라, 교회 실무 문서를 빠르게 완성하기 위한 간단 작성기입니다.
- 대부분의 경우 내용만 바꾸고 바로 PDF/PNG로 저장하면 됩니다.

## BASIC 1.0.5 수정

- `고급 페이지 설정` 표현을 `문서가 잘릴 때만 사용`으로 변경했습니다.
- 페이지 나눔 관련 안내문을 초보자 기준으로 쉽게 정리했습니다.
- `이 섹션부터 새 페이지`, `숨김` 같은 표현을 더 쉬운 문구로 바꾸었습니다.
- 간편 보기 상단에 `샘플 초기화` 버튼을 추가했습니다.
- `샘플 불러오기` 표현을 `샘플 초기화`로 통일했습니다.

## BASIC 1.0.4 수정

- 외부 표기는 `BASIC 1.0 작성기`로 통일했습니다.
- 판매용 첫 문구를 `10분 안에 완성` 중심으로 강화했습니다.
- 대표 OG 이미지를 상품 소개용으로 개선했습니다.
- PDF/PNG 저장 버튼명을 명확하게 정리했습니다.

## 기본 사용법

1. 왼쪽에서 만들 문서를 선택합니다.
2. 문서 편집판에서 내용을 입력합니다.
3. 오른쪽 미리보기에서 A4 출력 상태를 확인합니다.
4. 필요하면 간단 설정에서 글자 크기와 문서 밀도를 조정합니다.
5. `PDF 저장` 또는 `PNG 저장`을 누릅니다.

## 문서가 잘릴 때

대부분은 그대로 사용하면 됩니다.  
PDF 저장 시 특정 내용이 잘리거나 다음 페이지에서 시작해야 할 때만 `문서가 잘릴 때만 사용` 메뉴를 열어 조정하세요.

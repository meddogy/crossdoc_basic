# 교회문서키트 BASIC 1.0 작성기

교역자와 교회 실무자가 자주 쓰는 문서를 10분 안에 작성하고 PDF/PNG로 저장할 수 있는 교회 실무 문서 작성기입니다.

> 내부 관리 버전: BASIC 1.0.4

## 포함 문서 3종

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 교육부서 주간보고서

## 사용 흐름

문서 선택 → 내용 입력 → 미리보기 확인 → PDF/PNG 저장

## 구매자 안내 문구

반복되는 교회 문서 작성, 이제 10분 안에 끝낼 수 있습니다.
공지문, 월간행사 안내, 교육부서 주간보고서를 양식에 맞게 입력하고 바로 PDF/PNG로 저장하세요.

## BASIC 1.0.4 개선 내용

- 외부 노출명을 `교회문서키트 BASIC 1.0 작성기`로 통일했습니다.
- 내부 관리 버전은 1.0.4로 분리했습니다.
- 첫 화면과 OG 이미지 문구에 `10분 작성` 가치를 반영했습니다.
- 간단 설정 접힘 상태에서도 현재 글자 크기와 문서 밀도가 보이도록 개선했습니다.
- 버튼명을 초보자 기준으로 정리했습니다.
  - PDF 저장
  - PNG 저장
  - 샘플 불러오기
  - 빈 양식으로 시작
- 구매자에게 보일 README와 개발 수정 메모를 분리하기 쉽도록 구조를 정리했습니다.

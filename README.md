# 교회문서키트 BASIC 1.5

교회 실무자를 위한 문서 작성 웹앱입니다. 구매자 이메일 인증, PWA 설치 지원, PDF/PNG 저장 기능을 포함합니다.

## BASIC 1.5 주요 개선

- 행사 및 수련회 기획안 일정표 개선
  - 현재 선택한 일차에 일정을 추가하면 새 입력칸이 목록 맨 위에 바로 보입니다.
  - `일차 추가` 버튼으로 3박 4일 이상 일정도 쉽게 만들 수 있습니다.
  - 4박 5일, 5박 6일 프리셋을 추가했습니다.
- 부서 통합 주간보고서 개선
  - 부서별 현황에서 부서명을 직접 수정할 수 있습니다.
  - 예배부, 선교부, 속회, 소그룹 등으로 자유롭게 바꿀 수 있습니다.
  - 부서/팀 추가·삭제가 가능합니다.

## 기본 제공 문서

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 부서별 주간보고서
4. 부서 통합 주간보고서
5. 행사 및 수련회 기획안

## 배포 방법

1. GitHub 저장소에 전체 파일을 교체 업로드합니다.
2. Vercel에서 자동 배포가 완료되는지 확인합니다.
3. 기존 Supabase/Vercel 환경변수는 그대로 유지합니다.

## 환경변수

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## 사용 안내

개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다. 공용 PC에서 사용한 경우에만 `공용 PC에서 로그아웃`을 눌러 주세요.

# 교회문서키트 BASIC 1.16

교회 공지문, 월간행사 안내, 부서별·통합 주간보고서, 행사 및 수련회 기획안을 작성하고 PDF/PNG로 저장하는 교회 실무자용 웹 작성기입니다.

## BASIC 1.16 사용설명서 내장판

이번 버전은 판매·배포용 사용성을 더 보강했습니다.

- 상단과 빠른 메뉴에 `사용법` 버튼을 추가했습니다.
- 앱 안에서 바로 볼 수 있는 사용설명서 모달을 넣었습니다.
- 현재 선택한 문서에 맞는 문서별 사용 팁을 제공합니다.
- 로그인/바로가기/Sign in 링크 안내를 앱 안에서도 확인할 수 있습니다.
- 행사 및 수련회 기획안 일정표 편집판의 `1일차 일정 수정` 제목이 세로로 줄바꿈되지 않도록 보강했습니다.
- 기존 BASIC 1.15의 일정표 일차별 여유형, 예산 0원 처리, 하단 안내바 가림 방지, 표 글씨 크기 적용 보강을 유지합니다.

## 제공 문서 5종

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 부서별 주간보고서
4. 부서 통합 주간보고서
5. 행사 및 수련회 기획안

## 구매자 이메일 인증

Vercel 환경변수에 아래 값을 설정해야 합니다.

```txt
VITE_SUPABASE_URL
VITE_SUPABASE_ANON_KEY
```

Supabase의 `allowed_users` 테이블에 등록된 이메일만 작성기를 사용할 수 있습니다.

## 배포 방법

1. 압축을 풀고 전체 파일을 GitHub에 업로드합니다.
2. Vercel에서 자동 배포를 확인합니다.
3. 기존 Supabase/Vercel 환경변수는 그대로 유지합니다.
4. 배포 후 로그인하여 `사용법` 버튼, 일정표 편집판, PDF/PNG 저장을 확인합니다.

## 구매자 안내 핵심 문구

- 메일의 Sign in 링크는 로그인용 임시 링크입니다.
- 계속 사용할 주소는 작성기 기본 주소입니다.
- 개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다.
- 공용 PC에서는 `공용 PC에서 로그아웃`을 눌러 주세요.

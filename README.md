# 교회문서키트 BASIC 1.11

교회 실무자를 위한 문서 작성 웹앱입니다. 구매자 이메일 인증, PWA 설치 지원, PDF/PNG 저장 기능을 포함합니다.

## BASIC 1.11 판매 전 안정화판 주요 개선

### 1. 행사 및 수련회 기획안 일정표 안정화
- 편집판에서는 일정 카드가 **추가한 순서대로 유지**됩니다.
- 시작/종료 시간을 바꿔도 카드가 갑자기 위아래로 이동하지 않습니다.
- PDF/PNG 미리보기 일정표에서는 일정이 **시간순으로 정리**됩니다.
- 일정 프리셋은 `당일 행사`, `1박 2일`, `2박 3일`, `3박 4일`까지만 보입니다.
- 4일차 이상은 `+ 일차 추가` 버튼으로 늘립니다.

### 2. 부서 통합 주간보고서 부서별 현황 안정화
- 부서명과 내부 저장값을 분리했습니다.
- 부서명을 한글로 입력하거나 변경해도 입력 중 글자가 사라지지 않도록 개선했습니다.
- 같은 부서명이 생겨도 기존처럼 데이터가 꼬일 가능성을 줄였습니다.
- 부서/팀 추가, 삭제, 이름 변경이 더 안정적으로 작동합니다.

### 3. 예산 입력 방식 단순화
- 수입계획/지출계획은 기본적으로 **금액 직접 입력** 방식입니다.
- 수량×단가가 필요한 경우에만 `상세 산출`을 열어 사용할 수 있습니다.
- 수량을 비워 두면 합계가 0원으로 잡혀 헷갈리는 문제를 줄였습니다.
- 출력용 예산 표는 `항목 / 산출 내용 / 금액 / 비고` 4열로 단순화했습니다.

### 4. PWA 캐시 버전 정리
- 서비스워커 캐시 이름을 BASIC 1.11 기준으로 변경했습니다.
- 설치형으로 사용할 때 이전 버전 화면이 남아 보일 가능성을 줄였습니다.

### 5. 안내 문구 정리
- 시간은 직접 입력이 아니라 선택 방식입니다.
- 메일의 Sign in 링크는 임시 로그인용입니다.
- 저장해야 할 주소는 작성기 기본 주소입니다.
- 개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다.

## 기본 제공 문서

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 부서별 주간보고서
4. 부서 통합 주간보고서
5. 행사 및 수련회 기획안

## 배포 방법

1. GitHub 저장소에 전체 파일을 교체 업로드합니다.
2. Vercel에서 자동 배포가 완료되는지 확인합니다.
3. 기존 Supabase/Vercel 환경변수는 그대로 유지합니다.

## 환경변수

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## 사용 안내

개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다. 공용 PC에서 사용한 경우에만 `공용 PC에서 로그아웃`을 눌러 주세요.

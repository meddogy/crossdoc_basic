# 교회문서키트 BASIC 1.12

교회 실무자를 위한 문서 작성 웹앱입니다. 구매자 이메일 인증, PWA 설치 지원, PDF/PNG 저장 기능을 포함합니다.

## BASIC 1.12 최종 편의성 보강판 주요 개선

### 1. 일정표 시간순 정리 버튼 추가
- 편집 중에는 일정 카드가 자동으로 위아래 이동하지 않습니다.
- 필요할 때 `현재 일차 시간순 정리` 버튼을 눌러 해당 일차 일정만 정돈할 수 있습니다.
- 새 일정 추가 후 시간을 선택하고, 마지막에 정리 버튼을 누르는 흐름으로 사용합니다.

### 2. 부서 통합 주간보고서 부서명 입력 개선
- 부서명을 지웠을 때 자동으로 `부서1` 같은 값이 생기지 않도록 수정했습니다.
- 부서/팀명은 입력 중에는 그대로 유지되고, 다른 칸을 누르면 반영됩니다.
- 부서명을 완전히 지운 상태에서 새 이름을 다시 입력하기 쉬워졌습니다.

### 3. 예산 항목 접기/펼치기 정리
- 수입·지출 예산안의 4열 출력 형식은 유지했습니다.
- 입력 항목이 많아질 때 보기 힘들지 않도록 예산 항목을 접기/펼치기 카드로 바꿨습니다.
- 항목명, 산출 내용, 금액 요약을 먼저 보고 필요한 항목만 펼쳐 수정할 수 있습니다.
- 기본은 금액 직접 입력이며, 필요할 때만 `수량×단가 상세 산출`을 열어 사용합니다.

### 4. PWA 캐시 버전 정리
- 서비스워커 캐시 이름을 BASIC 1.12 기준으로 변경했습니다.
- 설치형으로 사용할 때 이전 화면이 남아 보일 가능성을 줄였습니다.

## 기본 제공 문서

1. 기본 공지 안내문
2. 각부 월간행사 안내
3. 부서별 주간보고서
4. 부서 통합 주간보고서
5. 행사 및 수련회 기획안

## 배포 방법

1. GitHub 저장소에 전체 파일을 교체 업로드합니다.
2. Vercel에서 자동 배포가 완료되는지 확인합니다.
3. 기존 Supabase/Vercel 환경변수는 그대로 유지합니다.

## 환경변수

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## 사용 안내

개인 PC에서는 로그아웃하지 않고 창만 닫아도 됩니다. 공용 PC에서 사용한 경우에만 `공용 PC에서 로그아웃`을 눌러 주세요.
